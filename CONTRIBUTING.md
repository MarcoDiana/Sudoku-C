# Come Contribuire

Siamo felici che tu voglia contribuire al progetto Sudoku! Segui queste semplici linee guida per iniziare.

## Setup del Progetto

1. Fai un fork della repository.
2. Clona il tuo fork localmente: `git clone <url-del-tuo-fork>`
3. Crea un nuovo branch per la tua feature o fix: `git checkout -b feature/la-mia-feature`

## Regole per il Codice

- Segui lo stile di formattazione esistente.
- Crea dei commit atomici per ogni modifica logica (es. `feat: aggiunge funzione X`, `fix: risolve bug Y`).
- Assicurati che il codice sia compilabile tramite CMake prima di aprire una Pull Request.

## Apertura della Pull Request

- Spiega chiaramente cosa cambia la tua PR.
- Fai riferimento a eventuali Issue aperte collegate al tuo lavoro.
